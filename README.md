# 共享公寓管理后台(前端)

## 体验

https://yinjiangqaq.com/sharedHouse/#/login

## 使用

克隆项目到本地

```
git clone https://github.com/yinjiangqaq/sharedHouse.git
```

安装依赖并运行项目

```
npm install

npm run start
```

项目会运行在 3000 端口，打开浏览器，地址栏输入：`http://localhost:3000`便可看到我们的项目

## 技术栈

React Hooks + ant design + axios

## 功能列表
